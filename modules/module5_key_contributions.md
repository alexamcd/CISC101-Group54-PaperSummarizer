Module 5: Key Contributions Extractor
Purpose:

Identify and present the paper’s core contributions, ensuring all claims are explicitly supported by the original text.

Responsibilities:

Detect explicit statements of contributions

Infer contributions only when text directly supports them

Distinguish major vs minor contributions

Produce expert + lay versions

Inputs:

Unified summary

Section summaries

Raw text (if needed for verification)

Outputs:

List of major contributions

List of secondary insights

Expert-mode contribution summary

Layperson-mode contribution summary

Rules & Constraints:

No invented contributions

Must verify textual support

Must preserve hierarchy (major vs minor)

Internal Steps:

Scan introduction, discussion, conclusion for contribution language

Extract explicit contribution claims

Check against methods/results for support

Produce contribution list

Rewrite list for expert + lay audiences

Failure / Warning Conditions:

Contribution cannot be verified → flagged
